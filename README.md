## ✨ Features

- 🌙 Dark/Light mode with persistent preference
- 🎨 Unique "Aurora" design — animated orbs, star particles, gradient text
- 📱 Mobile-first responsive design
- ⌨️ Keyboard shortcuts (`⌘K` focus input, `⌘/` toggle theme)
- 📤 Share buttons on every tool (X/Twitter, copy link)
- 🕐 Recently used tools (localStorage)
- 🔒 All processing happens in your browser — nothing sent to servers
- ♿ Accessible — focus visible, semantic HTML, skip-to-content

## 🏗 Tech Stack

- **Vite** — lightning-fast dev server and build
- **Vanilla JS** — zero frameworks, zero bloat
- **Vanilla CSS** — custom design system, no Tailwind
- **Google Fonts** — Inter + JetBrains Mono

Total bundle size: **~30KB** gzipped.

## 📦 Getting Started

```bash
# Clone
git clone https://github.com/mikalai-amusin/toolpilot.git
cd toolpilot

# Install
npm install

# Dev server
npm run dev

# Build for production
npm run build
```

## 🚢 Deploy (Free)

```bash
# Vercel (recommended)
npx vercel

# OR Netlify
npx netlify-cli deploy --prod
```
